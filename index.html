Mengenal Saya Lebih Dekat

IT Network System Admin & Full-Stack Engineer dengan keahlian lintas domain — dari infrastruktur jaringan hingga pengembangan aplikasi web.

Saya adalah Rochmad Djojo Oktabianto, biasa dipanggil Jojo — seorang IT Network System Admin sekaligus Full-Stack Engineer yang saat ini menempuh D4 Teknik Informatika di Politeknik Elektronika Negeri Surabaya (PENS).

Di bidang jaringan, saya berpengalaman dalam konfigurasi dan administrasi infrastruktur jaringan, manajemen server, serta troubleshooting sistem. Kemampuan ini saya buktikan melalui prestasi di kompetisi ITNSA (IT Network System Administration) tingkat kabupaten, provinsi, dan nasional.

Di sisi lain, saya juga aktif sebagai Full-Stack Engineer — merancang dan membangun aplikasi web dari frontend hingga backend. Kombinasi keahlian di bidang jaringan dan pengembangan perangkat lunak membuat saya mampu melihat solusi teknologi secara menyeluruh. Ke depan, saya aktif mengeksplorasi Data Science untuk melengkapi keahlian teknis yang saya miliki.
